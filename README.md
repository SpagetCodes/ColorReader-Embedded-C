# ColorReader-Embedded-C
Basic coloreader that uses a I2C for display and sensor TCS34725

This project is being developed as part of a class assignment, with an expected completion date at the end of March. Future updates will include a custom PCB, circuit schematics, and STL files for a compact, handheld version.
The platfrom is Altmega328P microcontroller.
Method used for bugfixes and troubleshooting is Amtel-ICE acc-basic debugger.
All feedback is welcome and much appreciated!
